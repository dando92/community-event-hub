# Community Event Hub

Community Event Hub is a platform for organizing, tracking, and preserving community-driven rhythm game events.

The project is not only a tournament manager.

The long-term goal is to build a shared hub where the community can manage different kinds of competitive and social events, including irl tournaments, online tournaments, pack release events, seasonal events, monthly events, local community events, and historicize all event data.

## Vision

Community Event Hub should become the place where all events are tracked.

The platform should preserve:

- who participated
- what was played
- which songs, charts, or packs were involved
- results
- how rankings changed

The platform should support both structured competitions and lighter community-driven activities.

The platform should support multiple event formats:

- irl tournaments
- online tournaments
- pack release events
- monthly events
- seasonal events
- community challenges
