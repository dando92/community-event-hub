# Community Event Hub

Community Event Hub is a platform for organizing, tracking, and preserving community-driven ITGMania and Deadsync events.

The long-term goal is to build a shared hub where the community can manage different kinds of competitive and social events, including IRL tournaments, online tournaments, pack release events, seasonal events, monthly events, and local community events. The platform should also preserve the history and data of every event.

## Vision

Community Event Hub should become the central place where community events are tracked.

The platform should preserve:

* who participated
* what was played
* which songs, charts, or packs were involved
* event results
* ranking history

The platform should support both structured competitions and lighter community-driven activities.

The platform should support multiple event formats:

* IRL tournaments
* online tournaments
* pack release events
* community events
* seasonal events
